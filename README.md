# Space Invaders Game — Python (Pygame)

A simple Space Invaders game built using Python and Pygame.

## Overview
This project implements a classic Space Invaders–style game to practice game logic, modular programming, and event handling using Python.

## Features
- Player movement and shooting
- Enemy movement and collision detection
- Score tracking
- Modular code structure

## Technologies Used
- Python
- Pygame

## How to Run
```bash
pip install pygame
python main.py
